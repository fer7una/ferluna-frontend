# Fernando Luna Frontend

Aplicación React + Vite para el portal personal de Fernando Luna.

## Ejecutar

```powershell
npm install
npm run dev
```

La API se lee desde `VITE_API_BASE_URL`. Si no existe, usa `http://127.0.0.1:8000`.

```env
VITE_API_BASE_URL=http://127.0.0.1:8000
```

## Scripts

- `npm run dev`
- `npm run build`
- `npm run preview`

## Extender el portal público

Consulta `docs/public-portal-extension.md` antes de añadir secciones, pestañas
momentáneas, variantes de card o animaciones nuevas.
